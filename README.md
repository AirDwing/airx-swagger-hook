# airx-swagger-hook

## 使用

在 Swagger.html 中加入：

```html
<script src="//unpkg.com/jssha"> </script>
<script src="//unpkg.com/@airx/authcode/dist/authcode.js"> </script>
<script src="//unpkg.com/@airx/swagger-hook"> </script>
```

p.s. 仅限于`AirX`, `QCloud` 项目和相同加密方式使用。
